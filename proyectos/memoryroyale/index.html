Sobre el proyecto

Descripción

Memory Royale es un juego de memoria 2D desarrollado en Unity como Proyecto Final del primer curso del ciclo DAM (Desarrollo de Aplicaciones Multiplataforma).

El jugador debe encontrar todos los pares de cartas en el tablero dentro del menor número de intentos posible, con animaciones fluidas implementadas con DOTween.

Tecnologías
Unity C# DOTween ScriptableObjects WebGL Fisher-Yates State Machine Prefabs
